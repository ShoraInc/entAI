# 🎓 ЕНТ Информатика Боты

AI-ассистент для подготовки к ЕНТ по Информатике. Общается на русском и казахском языках.

---

## ⚡ Установка (3 шага)

### 1. Установи Node.js
Скачай с https://nodejs.org (выбери LTS версию)

### 2. Установи зависимости
Открой папку с ботом в терминале и выполни:
```bash
npm install
```

### 3. Заполни .env файл
Открой файл `.env` и вставь свои ключи:
```
TELEGRAM_TOKEN=твой_токен_от_BotFather
OPENAI_API_KEY=твой_новый_openai_ключ
```

> ⚠️ ВАЖНО: Старый OpenAI ключ нужно сбросить на https://platform.openai.com/api-keys

---

## ▶️ Запуск
```bash
node bot.js
```

Бот будет работать пока открыт терминал.

## ❓ Частые вопросы

**Бот не отвечает?**
- Проверь что токены правильно вставлены в `.env`
- Убедись что `npm install` выполнен
- Проверь интернет соединение

**Ошибка 429?**
- Превышен лимит OpenAI. Подожди минуту.

**Как остановить бота?**
- Нажми `Ctrl + C` в терминале
